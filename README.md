# Exploring London the Green Way – eXeLearning Project

## 📘 How to Open and Use This Project

This project has been created using **eXeLearning** and exported as a complete HTML website.

To correctly view and navigate the project, please follow these instructions:

### ✅ Option 1: Open Locally (Recommended)

1. Download the ZIP file.
2. Extract (unzip) the folder on your computer.
3. Open the extracted folder.
4. Double-click on the file **`index.html`**.
5. The project will open in your web browser.

⚠️ Important:

* Do **not** move or delete any files inside the folder.
* The `index.html` file must remain in the same folder as all other files (content, libs, theme, etc.), otherwise the project may not display correctly.

---

### 🌐 Option 2: View Online

If the project has been uploaded to an online platform (such as GitHub Pages or Moodle), simply open the provided link in your browser and navigate through the sections.

---

## 📂 Project Structure

The folder includes:

* `index.html` → Main file to open the project
* `content/` → Learning content
* `libs/` → Technical resources
* `theme/` → Visual design
* Other supporting files

All these elements are necessary for proper functioning.

---

## 🎯 About the Project

**Title:** Exploring London the Green Way
**Level:** 5th Grade Primary Education
**Subject:** English as a Foreign Language

Students design a **sustainable one-day itinerary in London**, using English to describe landmarks, give directions, and explain eco-friendly transport choices.
